# 美识谱
#### 介绍
这是一款食谱 app ，一款可以拍照识谱，个性化推荐食谱的便民食谱应用。此应用程序实现了用户浏览食谱，给
用户推荐食谱，用户上传自制菜谱，智能识别菜品，用户私信与管理员审核等功能。
